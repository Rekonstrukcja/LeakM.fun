LeakM.fun
1. Otwieracie Visual studio code, następnie klikacie "Open Folder" i wybieracie folder tam gdzie macie dumpa. 2. Przechodzicie do lupy i szukacie w niej słów związanych z tym czego szukacie. 3. Po znalezieniu tego szukacie powinniście mieć "vec4(kordy)" lub "coords = (kordy)". 4. Kopiujecie te kordy i odpalacie GTA XYZ (strona) lub Code Walker (program). 5. Jeżeli odpaliliście GTA XYZ to na tej stronie wpisujecie 2 pierwsze koordynaty (X i Y), następnie pokaże wam się punkt na mapie w którym owe miejsce jest. 6. Jeżeli odpaliliście Code Walker (służy do bardziej precyzyjnego sprawdzania koordynatów) 7. W Code Walker, klikacie << a następnie przechodzicie do Markers. Poddajecie pełne koordynaty (x, y, z). 8. Zaznaczacie Show marker, następnie klikacie GO TO i przenosi was do miejsca. ALTERNATYWNA OPCJA: Pracuje nad programem który sam szuka najistotniejszych rzeczy i wypisuje a ty sprawdzasz w visual studio code ich poprawność.
